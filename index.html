Coffee For Coders

MEAN Stack Application (Group project)

Built with: HTML, CSS, Javascript, Mongo, Express, Angular, Node.js, Mongoose, and RESTfulrouting.

Team members: Sarah Anderson (Colorado) and Rick Neely (Georgia)

Description: Coffee For Coders is a single page application where users can search for coffeeshops in their city. They can add a coffeeshop to the list of top coffeeshops in the U.S. We used an MVC file structure with 2 models, one of which has full CRUD capabilities. We used Sockets.io to create a chat box, so you can talk to other people who are on the site. We used bcrypt and Express Session for user authentication.

My Contributions:

  • Merged GitHub branches after commits
  • Helped to solve merge conflicts
  • Linked the models
  • Implemented the Yelp API
  • User authentication and session
  • Added session logic, so that only a logged in user can see the add, edit, or delete coffeeshops buttons/forms
  • Made sure forms were removed after submitting

Fantasy Football Comparison Tool

Ruby on Rails Application (Partner project)

Built with: HTML, CSS, Javascript, Express, Angular, Node.js, Ruby, Rails, SQL, and RESTfulrouting.

Partner: Mike Skelton (Pennsylvania)

Description: You can use this Fantasy Football Comparison Tool to compare the stats of two different players to see who you should draft/start for your fantasy football team. We built the API using Ruby on Rails and SQL, and the front end was built using JS, Angular, Node.js, and Express. There are two models with full CRUD operations available. The models have a many to many relationship.

My Contributions:

  • Made AJAX requests to the API
  • Deployed CORS
  • Created the players' filter function
  • CSS
  • Made the site responsive
  • Added create, update, and delete functionality to the users model
  • Deployed front/backend to Heroku
  • Resolved merge conflicts
